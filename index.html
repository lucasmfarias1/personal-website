DuoQ

Web app que realiza matchmaking de jogadores de League of Legends para jogarem em dupla (duo) nos moldes de dating apps como o Tinder. Utiliza OAuth através do Facebook, com Devise, para autenticação. Logo e design feitos em parceria com designer gráfica.

Stack: Ruby, Ruby on Rails, HTML, CSS, JavaScript e jQuery

SoloKill

Web app desenvolvido para ser um Twitter clone que usa a API da Riot Games para integrar a conta do usuário com sua conta do jogo League of Legends e representar o usuário com seu nome de jogador. Utiliza OAuth através do Facebook, com Devise, para autenticação.

Stack: Ruby, Ruby on Rails, HTML, CSS, JavaScript e jQuery
